Over mij

Welkom op mijn portfoliowebsite! Via deze website laat ik graag mijn projecten, zowel vroegere als huidige, aan geïnteresseerden zien!

Ik ben Diko van der Meer, een beginnend Front-End Developer met een achtergrond in design. Ik leer graag nieuwe dingen en zie dit ook als een uitdaging. Als je in aanraking komt met de verschillende denkwijzen die developers gebruiken om problemen op te lossen, kun je tot nieuwe inzichten komen. Ik ga dan ook graag hierover met andere developers in gesprek.

Ook ben ik geïnteresseerd in activiteiten die met design of development te maken hebben, zoals game-ontwerp, game-development en tekenen.

Het liefst maak ik dingen die voor mensen leuk óf nuttig zijn
(beide mag natuurlijk ook!) en wil ik hier in blijven groeien.

Gemaakte Projecten

Projecten die ik heb gemaakt, tijdens het leren van Front-End Development of gewoon voor mijzelf.
Sommigen Projecten maakten gebruik van API's die misschien niet meer werken. Ik heb voor de demo's deze projecten hiervan losgekoppeld. Via de bijbehorende Github-link kan de originele code nog wel worden bekeken.

De meeste, maar niet alle projecten zijn gemaakt rekening houdend met mobiele devices.
Voor de meest universele experience met de demo's raad ik aan grotere schermen te gebruiken

  • Eindopdracht Winc Academy

    Een studenten-dashboard in React. Informatie over studenten en beoordelingen die zij aan opdrachten hebben gegeven kunnen worden bekeken met behulp van verschillende filteropties.

    Demo Github

  • Liedjeslijst

    Stelt de gebruiker in staat om zelf liedjes aan een lijst toe te voegen en deze lijst te sorteren en filteren.

    Demo Github

  • Film Finder

    Een gebruikersinterface waar de gebruiker filter- en zoekopties kan gebruiken om films te vinden in een database. De website geeft de optie door te klikken naar de bijbehorende IMDB-pagina.

    Demo Github

  • Kleurenswitch

    Een simpele webapplicatie welke de gebruiker in staat stelt met een menu of de toetsen 1 t/m 5 de kleur van de pagina te veranderen. Een oefening in het combineren van JavaScript en CSS.

    Demo Github

  • Geavanceerde CSS

    Een geavanceerde CSS/SCSS demo, waarbij een gegeven design moest worden nagemaakt. De nadruk lag vooral op grid- en flexbox-layouts en het 'responsive' maken van de pagina.

    Demo Github

  • Simpele HTML/CSS Website

    Een simpele website met meerdere pagina's over een onderwerp wat ik interessant vind.

    Demo Github
